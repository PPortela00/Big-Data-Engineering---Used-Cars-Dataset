O projeto está organizado por pastas para ser mais fácil o acesso a todos os seus componentes.

	- notebook-outputs: contém todos os outputs que foram obtidos no notebook. Cada etapa é um ficheiro diferente

	- notebook-to-py: contém todos os ficheiros .py que foram utilizados no dataproc

	- performance-csv: contém todos os .csv obtidos na análise do cluster

	- performance-outputs: contém todos os outputs obtidos nos testes efetuados no dataproc, bem como o notebook utilizado para efetuar as respectivas visualizações

	- performance-py: contém todos os ficheiros .py que foram utilizados para efetuar os testes no dataproc

	- plots: contém todos os gráficos gerados ao longo do projeto

	- Cars_final_notebook: notebook com a última versão de todo o código
